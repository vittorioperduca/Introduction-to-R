Chapter 1 Welcome

This document is as a tutorial for self-learning the basic use of R. It was prepared for the students enrolled in the first year of the AIRE Life Sciences Master Program of the Centre de Recherche Interdisciplinaire at the Université de Paris.

Although you will find an explanation of the basic objects and commands, this introduction is not a complete reference to the R language, so you will have to use the help and search for information on the web. Before moving on to the exercises at the end of each chapter, you are encouraged to type the commands and understand the result: the best way to learn a language is to write and debug lots of lines of code!
